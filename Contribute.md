# Contributing to Recipes

Thanks for your interest in contributing! We welcome new recipes, improvements, and fixes.  

## How to Contribute

1. **Fork the repository** and clone it locally.  
2. **Create a new branch** for your changes:
   ```bash
   git checkout -b my-new-recipe
Add your recipe following the existing format in the recipes/ folder. Include:

Title

Ingredients

Instructions

Optional: Notes, Tips, or Photos

Commit your changes with a clear message:

git commit -m "Add Chocolate Chip Cookies recipe"
Push your branch to your fork:

git push origin my-new-recipe
Open a Pull Request describing your contribution.

Guidelines
Keep recipes clear and easy to follow.

Use consistent formatting.

Avoid duplicating recipes already in the repository.

Add images only if they are your own or free to use.

Code of Conduct
Be respectful and constructive in your contributions and discussions.

Thank you for helping make this repository better!
